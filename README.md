# 🎓 Student Feedback Analyzer

An AI-powered web application that analyzes student feedback and transforms raw responses into meaningful insights using Natural Language Processing (NLP) and Sentiment Analysis.

## 📌 Overview

Educational institutions collect large amounts of student feedback every semester. Manually analyzing this feedback can be time-consuming and inefficient. This project automates the process by identifying sentiments, extracting key insights, and presenting the results through interactive visualizations.

## 🚀 Features

* 📂 Upload student feedback datasets
* 😊 Sentiment Analysis (Positive, Neutral, Negative)
* 🔍 Keyword Extraction
* 📊 Interactive Data Visualizations
* 📈 Feedback Statistics and Trends
* ⚡ Real-Time Analysis
* 🎯 User-Friendly Interface
* 📝 Insight Generation for Better Decision Making

## ⚙️ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/yourusername/student-feedback-analyzer.git
cd student-feedback-analyzer
```

### 2. Create Virtual Environment

```bash
python -m venv venv
```

### 3. Activate Environment

Windows:

```bash
venv\Scripts\activate
```

Mac/Linux:

```bash
source venv/bin/activate
```

### 4. Install Dependencies

```bash
pip install -r requirements.txt
```

### 5. Run the Application

```bash
python run app.py
```

## 📊 How It Works

1. Upload a student feedback dataset.
2. The application processes the feedback text.
3. Sentiment analysis is performed.
4. Important keywords are extracted.
5. Results are displayed through charts and visualizations.
6. Users gain actionable insights from student responses.
